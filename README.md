# Oleksandr Brynzei - Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing the work and experience of Oleksandr Brynzei, a Senior Front-End Developer.

## About

This portfolio site presents a comprehensive overview of professional experience, skills, projects, and education. The design emphasizes clean aesthetics and smooth user experience to highlight the developer's work and expertise.

## Tech Stack

- **React** - UI library for building the interface
- **Vite** - Fast build tool and development server
- **CSS** - Styling and responsive design
- **JSON** - Content management for portfolio data

## Features

- **Hero Section** - Introduction with professional title and background
- **About** - Personal biography and professional overview
- **Skills** - Technical expertise across languages, frameworks, and tools
- **Experience** - Career timeline with detailed role descriptions
- **Projects** - Curated selection of recent work with live links
- **Education** - Academic background and credentials
- **Testimonials** - Recommendations from colleagues and clients
- **Contact** - Get in touch information

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

### Build

To create a production build:

```bash
npm run build
```

### Preview

To preview the production build:

```bash
npm run preview
```
